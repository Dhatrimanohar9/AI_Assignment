# Turing Test and CAPTCHA

## Turing Test

The Turing Test was proposed by Alan Turing to determine whether a machine can behave like a human.

In this test, a human judge communicates with both a human and a computer through a text interface.  
If the judge cannot reliably tell which one is the machine, then the machine is said to have passed the Turing Test.

Architecture:

Human Judge → Interface → Computer Agent / Human Participant

Purpose:
To evaluate whether a machine can demonstrate intelligent behavior similar to humans.

---

## CAPTCHA

CAPTCHA stands for:
Completely Automated Public Turing test to tell Computers and Humans Apart.

It is used on websites to check whether the user is a human or a bot.

Examples include:
- Distorted text recognition
- Image selection (select traffic lights)
- Checkbox verification

Architecture:

User → CAPTCHA System → Verification → Access Granted or Denied

Purpose:
To prevent automated bots from accessing websites or submitting forms.
